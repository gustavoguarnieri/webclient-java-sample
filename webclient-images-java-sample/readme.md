# Web Client Consumer Java Sample

The project aims to expose image rest resources to be consumed.

## Requirements

Java 11+

IntelliJ IDEA / Netbeans / Eclipse

## Usage

```python
getImages:
curl --location --request GET 'http://localhost:9081/api/v1/products/3/images'
```
